# Project-Virulence
A lightweight and optimized survival horror modpack for minecraft centered around Scape and Run Parasites

# How to create an instance
1. Download Prism Launcher, and log in: https://prismlauncher.org/
2. Create a new instance and set the version to 1.12.2, and select forge as the mod loader, pick the version at the top.
3. Once you finish making the instance, download packwiz-installer-bootstrap: https://github.com/packwiz/packwiz-installer-bootstrap/releases/tag/v0.0.3
4. Edit instance, go to settings and add a this prelaunch command: "$INST_JAVA" -jar packwiz-installer-bootstrap.jar https://raw.githubusercontent.com/HydratedBean/Project-Virulence/main/pack.toml
5. Press launch. Good luck, have fun.
