## A lightweight and optimized survival horror modpack for minecraft centered around Scape and Run Parasites

## How to create an instance
1. Download Prism Launcher, and log in: ```https://prismlauncher.org/```
2. Create a new instance and set the version to ```1.12.2```
3. Select forge as the modloader and pick version ```14.23.5.2860```
4. Once you finish making the instance, download packwiz-installer-bootstrap from here: ```https://github.com/packwiz/packwiz-installer-bootstrap/releases/tag/v0.0.3```
5. Click edit instance, go to settings, then custom commands
6. Checkmark custom commands and add a this prelaunch command:
   ```"$INST_JAVA" -jar packwiz-installer-bootstrap.jar https://raw.githubusercontent.com/HydratedBean/Project-Virulence/main/pack.toml```
8. Installation is complete. Press launch and have fun!
